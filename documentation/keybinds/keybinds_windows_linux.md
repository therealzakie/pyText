# Keybinds (Windows/Linux)

- Ctrl + N - New file 
- Ctrl + O - Open file 
- Ctrl + S - Save file 
- Alt + S - Save as file 
- Ctrl + D - Discard file 
- Ctrl + C - Copy selected text 
- Ctrl + X - Copy selected text and remove it 
- Ctrl + V - Pastes copied text 
- Ctrl + A - Select all text in document 
- Ctrl + / - Open pyText settings 
- Ctrl + W - Close pyText
