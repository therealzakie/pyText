# Keybinds (macOS)

- Cmd + N - New file 
- Cmd + O - Open file 
- Cmd + S - Save file 
- Opt + S - Save as file 
- Cmd + D - Discard file 
- Cmd + C - Copy selected text 
- Cmd + X - Copy selected text and remove it 
- Cmd + V - Pastes copied text 
- Cmd + A - Select all text in document 
- Cmd + / - Open pyText settings 
- Cmd + W - Close pyText
